Implementation of Graph Database for OpenCog Artificial General Intelligence Framework using Neo4j

Hendy Irawan, Ary Setijadi Prihatmanto

September 3, 2015

Abstract

We create a graph backing store[1] for OpenCog[2, 3] using Neo4j graph database[4]. The GraphBackingStore API will extend the current BackingStore C++ API. It can take special queries that map naturally into Cypher queries and simple manipulations of Neo4j graph traversals.

The Neo4j node-relationship structures and custom indices are optimized for AtomSpace[5] usage and performance. Data access is using Protocol Buffers-based[6] protocol over ZeroMQ[7] messaging transport.

The initial work for Neo4j Graph Backing Store for OpenCog was implemented during Google Summer of Code 2015. All source code is open source and available at https://github.com/ceefour/opencog-neo4j, https://github.com/opencog/atomspace, and https://github.com/opencog/opencog.
